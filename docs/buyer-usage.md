# Buyer Usage Guide

## Normal workflow

1. Open Shopee Seller Centre and make sure you are logged in.
2. Open the extension popup.
3. Click **Start**.
4. Wait for the current batch to finish.
5. Click **Export Excel** or **Export CSV**.

## Notes

- The extension refreshes the seller page before syncing.
- The license is bound per store, not per individual browser user.
- If multiple team members use the same store, the same license can work for them.
- If the store changes, the license server checks the allowed store limit.

## If something fails

- keep the Shopee store page open
- click **Start** again
- if activation or export still fails, send the popup screenshot and store name to support
