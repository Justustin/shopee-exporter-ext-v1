# Buyer Install Guide

## What you need

- Google Chrome
- access to Shopee Seller Centre
- the extension package from the seller
- your activation key

## Install steps

1. Save and unzip the extension package.
2. Open `chrome://extensions`.
3. Turn on **Developer mode**.
4. Click **Load unpacked**.
5. Select the unzipped extension folder.

## Activate

1. Log into `https://seller.shopee.co.id`.
2. Open the extension popup.
3. Paste your activation key.
4. Click **Activate License**.

## First use

1. Open the Shopee finance page.
2. Click **Start**.
3. Wait until the popup shows the sync is ready.
4. Export Excel or CSV.
